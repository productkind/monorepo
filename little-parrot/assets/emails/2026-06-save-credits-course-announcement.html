🛠️ Set up your builder's workspace

In a few steps, you'll have the same free tools developers use running on your own computer. And this isn't a one-off for this app. It's the workspace you'll use for every app and every document you work on with AI from here on.

💸 Stop spending credits on small fixes

A quick typo fix or an icon swap shouldn't cost you a thing. Once you can make those changes yourself, you'll know exactly when to do it by hand and when Lovable is worth the credits, so your credits go to the work that really needs them.

🤖 Hand the bigger changes to Copilot

Got a bigger change in mind? You'll describe it to GitHub Copilot, watch it do the work for free, and look it over before saving it back to GitHub and Lovable. Every version is saved along the way, so you can roll back any time without worrying about breaking your app.

Built around Lovable, but it works just as well if you build with Replit, v0, or Bolt. They all connect to GitHub.
