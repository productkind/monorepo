## Tips for capturing

- Use Chrome at standard zoom (100%)
- Light theme (shadcn default)
- Screenshot only the component preview box (the first example on each page), not the full page
- Aim for consistent width (~600px) so they look uniform in the guide
- For interactive components (Dialog, Drawer, Toast, Context Menu, Command), click to open the component first, then screenshot the open state
- File Upload has no dedicated shadcn page; screenshot a typical dropzone from a Lovable-built app or use a placeholder
