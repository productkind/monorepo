# Instagram: Build Your First App with Lovable (7 carousels)

Seven Instagram carousels built from the **Build Your First App with Lovable**
course (`lovable-intro-00`) and its toolkits, shaped by
`productkind/marketing/strategy/audience-research.md` and the content strategy.

## Why these six (research trace)

- **Six of the seven repackage existing toolkit items** (prompt template,
  prompt library, Project Knowledge template, pre-publish checklist, feedback
  script), per the reuse-first Instagram rule: pre-made saveable content.
- **Saveable formats win saves**, the metric that matters: every carousel is a
  recipe, checklist, or decision guide someone keeps for her next session.
- **The set covers the whole learner journey**, including the messy middle
  (things breaking, reverting, starting fresh) that the research says nobody
  else covers and our audience names as their No. 1 pain.
- **This course is the builder wedge**: its first challenge is free and it's
  the free-course link paid users share, so every CTA points there.

## Posting notes

- Account: Little Parrot Instagram. Cadence 1 to 2 per week, so this set
  covers four to five weeks. Suggested order: 1 → 7 → 2 → 3 → 4 → 5 → 6.
  It mirrors the journey (first prompt → how to format it → recover → grow →
  consistency → publish → feedback); post 7 follows post 1 directly because
  the two are a natural pair (the formula gives the sections, the markdown
  guide gives the formatting).
- CTAs name the full course title (Build Your First App with Lovable) and
  LittleParrot.app, because the Instagram bio holds only 5 links and we have
  8+ courses, so "link in bio" doesn't scale. The UTM URL below remains for
  wherever the course link is actually placed (a bio slot, stories, comment
  replies):
  `https://littleparrot.app/5e86e580-264c-442c-8cc4-be5645f13e87/course-overview?utm_source=instagram&utm_medium=social&utm_campaign=build-first-app`
  (add `&utm_content=post-1` etc. per post for per-post tracking).
- The course's first challenge (Your First Prompt) is free; CTAs say so.
- Carousel slides: 1080 x 1350 (4:5 portrait), Little Parrot brand colours,
  large type, one idea per slide, same visual system as the
  `2026-july-instagram-save-credits-course` set so the account reads as one
  series.
