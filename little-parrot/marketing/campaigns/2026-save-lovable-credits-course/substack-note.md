# Substack Note - Little Parrot

Note from Kinga. New course announcement, value-first. Course overview link with `?ref=substack`.

---

Many of our learners ask us the same thing repeatedly: how do you make the small changes to your app yourself, instead of spending Lovable credits on every tweak?

It's a fair question. We build LittleParrot.app with a few tools at once. Some changes we make in Lovable, some we hand to Claude Code, some we do by hand in a code editor, then commit to GitHub so everything stays in sync.

The hard part is the setup. Doing this on your own computer means installing a handful of tools you've never opened before, and that list can feel like a wall before you've started. We didn't want that to lock anyone out.

So our newest course walks you through it in a kind way: setting up your builder's workspace, making a change by hand in a code editor, and using GitHub Copilot, which has a free tier you can start on today.

One thing becomes clear quickly: Editing with an AI agent that changes your files directly is far faster than copying answers back and forth from a chat assistant like ChatGPT or Claude.ai. And the same approach works on anything you write, not only code, so it's a first step to editing everything faster.

The course is "Save Lovable Credits: Edit Your App Like a Developer" on LittleParrot.app.
