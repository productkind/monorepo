# Instagram: Fix Bugs with Confidence (6 carousels)

Six Instagram carousels built from the **Fix Bugs with Confidence: Debugging
Your Lovable App** course (`vibe-coding-debugging-00`) and its toolkits,
shaped by `marketing/content-strategy/audience-research.md` and the content
strategy.

## Why these six (research trace)

- **This is the messy-middle flagship.** The doom loop ("the agent says it
  fixed it; it isn't fixed; credits drain") is our No. 1 user pain, the
  audience's own vocabulary ("fix and break cycle", "error loop"), and the
  biggest competitive gap: everyone sells the first hour, nobody teaches
  week 4. Every post here is that gap.
- **Diagnosis before fixing is the exact skill the audience asks for** in
  their own words ("Help me identify where the exact problem is so that I
  can prompt accordingly", r/lovable). Posts 2, 3, and 4 teach it directly.
- **Direct ROI framing:** every escaped bug loop is credits saved. One
  avoided doom loop pays for a month of Little Parrot.
- **Security has a canonical fear story** ("Guys, I'm under attack") and is
  the audience's own bar for "production ready real". Post 6 handles it
  calmly, framing the learner's situation rather than the tool's failings.
- **All six repackage existing toolkit items**, per the reuse-first
  Instagram rule, and every one is a saveable checklist, cheat sheet, or
  decision guide.

## Posting notes

- Account: Little Parrot Instagram. Cadence 1 to 2 per week, so this set
  covers three to four weeks. Suggested order: post in numbered order. Post 1
  (the bug loop escape) leads because it's the strongest pain hook; the rest
  follow the investigate → report → prevent → secure arc.
- CTAs name the full course title (Fix Bugs with Confidence: Debugging
  Your Lovable App) and say it's on LittleParrot.app, because the Instagram
  bio holds only 5 links and we have more courses than slots. Wherever the
  course link is actually placed (a bio slot, stories, comment replies),
  use:
  `https://littleparrot.app/aaab45b7-209a-4578-a515-3e55687f0c53/course-overview?utm_source=instagram&utm_medium=social&utm_campaign=fix-bugs`
  (add `&utm_content=post-1` etc. per post for per-post tracking).
- The course's first challenge (The Anatomy of a Bug) is free; CTAs say so.
- Carousel slides: 1080 x 1350 (4:5 portrait), same visual system as the
  other July sets.
- Overlap note: post 2 of the build-first-app campaign (fix, revert, or
  start fresh) is the light version of this campaign's post 1. They
  complement rather than repeat, but leave a week or two between them.
