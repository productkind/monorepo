# New course ➡ Save Lovable Credits: Edit Your App Like a Developer 💰

Hey Builders,

The #1 complaint we hear about Lovable is how fast credits are drained. Our new course shows you how to set up your builder's workspace, **make the small changes** yourself, on your own computer **for free**, and only call on Lovable when it's the necessary route.

## 💸 Stop spending credits on small fixes

A change you make and save yourself costs nothing. Asking Lovable to do it costs credits. By the end of this course you'll know exactly when to do which.

## 🛠️ Set up your builder's workspace

You'll set up the same free tools developers use (VS Code, Node.js, Git), run your app on your own computer, and watch every change appear the moment you make it. It works for every app, and any document, you'll build with AI from here on.

## 🤖 Hand the bigger changes to Copilot

For complex changes, you'll describe them to GitHub Copilot, review its work, then save it all back to GitHub and Lovable with a version history you can roll back any time.

Built around Lovable, but it works just as well if you build with Replit, v0, or Bolt. They all connect to GitHub.

Have fun with it, and tell us how you will use your new developer powers! 💛
